# Development Branch

This is the main development branch for the recruiting game project.

## Purpose
- Integration branch for all features
- Staging area before merging to main
- Contains latest development code

## Workflow
1. Create feature branches from this branch
2. Merge completed features back to this branch
3. Periodically merge to main for releases

## Current Features in Development
- Dynamic resume data generation
- Enhanced analytics tracking
- Timer functionality
- Tab viewing analytics
